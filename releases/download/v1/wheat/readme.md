# Wheat Disease Detection & AI-Based Diagnosis System
 

## Key Contributions

### 🔍 Dataset Research & Development
- Researched, collected, and curated a **large-scale Wheat Disease Dataset** with **14,155+ high-quality images**.
- Ensured dataset diversity across **multiple disease and pest categories** to improve model robustness and real-world performance.
- Used **high-resolution agricultural images** for better feature extraction and disease recognition.

### 🤖 AI Model Development
- Designed and trained a **CNN (Convolutional Neural Network)-based disease classification model** achieving **91% accuracy**.
- Enhanced prediction performance by integrating **Random Forest** and **Decision Tree classifiers** with **symptom-based user input**, enabling hybrid diagnosis.
- Improved classification reliability for visually similar diseases through multi-model inference.

### 📊 Real-Time Disease Identification Dashboard
- Built a **real-time disease identification dashboard** for quick and efficient diagnosis.
- Reduced disease diagnosis time significantly, enabling **faster farmer intervention**.
- Helped minimize **crop loss risk** through early detection and actionable insights.

### 🧠 AI-Powered Disease Insights
- Leveraged AI to provide **detailed disease analysis, symptom insights, and user feedback**.
- Improved diagnosis precision by combining **image recognition + symptom-based intelligence**.
- Enabled **timely agricultural interventions** through explainable AI-driven recommendations.
